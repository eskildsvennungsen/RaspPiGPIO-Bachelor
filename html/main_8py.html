main.CheckPhotoCell (   old_state,
  new_state 
)

Checks the photocell if a new bag has passed.

A new bag is considered to be in the system when it has fully passed the photocell. We check this by comparing the old_state to the new_state.

if old_state and old_state != new_state
Parameters
old_stateThe previous state of the photocell.
new_stateThe current state of the photocell.
Returns
(True, new_state) If bag has passed.
(False, new_state If bag has not passed.
